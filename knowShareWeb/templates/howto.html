StudyGate & BBB 이용방법


강의실 입장하는 방법

우선 회원가입하시고 로그인 하셔야 합니다.
1. 선생님 소개 바를 누릅니다.
2. 원하는 선생님의 강의실 입장 버튼을 누릅니다.


3. 자신의 대화명을 입력하고 강의실 입장 버튼을 눌러서 선생님 강의방에 입장하셔야 합니다.
만약에 위와 같은 화면이 아닌 "강의실에 입장할 수 있는 권한이 없습니다."라고 뜰 경우 관리자와 선생님과 상의해서 권한을 획득하셔야 합니다.


BBB 사용 방법

1. 허용 버튼을 눌르면 카메라를 사용할 수 있게 된다.
2. 다음에는 사운드 테스트 화면인데, 영어로 무슨 소리가 들린다면 Sound는 Ok! Default 바 뭔가 변화가 있다면 마이크도 동작하는 것이다.
Join Audio를 눌러서 오디오를 사용할 수 있게 한다.

1. 카메라 버튼을 누릅니다.
2. 카메라에 웹캠이 설치되어 있다면, 자신의 모습이 보이실 겁니다. 좌측 하단의 버튼을 눌르면 다른 사람에게 자신의 영상을 공유하게 됩니다.
3. 해당 화면에서 영상 공유를 한 사람들의 영상이 보이게 됩니다.


BBB 기능 설명

강의실 화면입니다. 일단 순서대로 설명드리겠습니다.
1. 3개의 버튼이 순서대로 있습니다. 첫번째 버튼은 화면 공유를 설정합니다. 두번째 버튼은 스피커와 마이크를 설정합니다. 세번째 버튼은 카메라를 통한 화면 공유를 설정합니다.
2. 현재 강의실에 입장한 사람에 대한 화면입니다. 연필을 들고 있는 학생 문양은 현재 칠판을 편집할 수 있는 권한을 가진 사람을 의미합니다.
3. 스피커를 켠 사람들에 대한 화면입니다.
4. 비디오 화면을 공유한 사람에 대한 화면입니다. 1번에서 3번째 버튼을 클릭후 재생을 누르면 내 화면을 다른 사람과 공유하게 됩니다. 그때 공유되는 화면은 이 창에서 뜹니다.
5. 슬라이드 공유 화면입니다.
6. 채팅창 화면입니다.

슬라이드 공유 화면만 확대해 놓은 부분입니다. 주요기능은 다음과 같습니다.
1. 강의자료를 올리는 부분입니다. PPTX와 PDF를 지원합니다.
2. 강의자료 슬라이드를 넘기거나 앞으로 가는 부분입니다.
3. 강의자료 슬라이드를 확대하거나 축소하는 부분입니다.
4. 좌측의 두버튼은 슬라이드 크기에 맞게 세로 맞춤, 전체크기 맞춤을 해줍니다. 우측 버튼을 누르면 5번과 같은 슬라이드 편집바가 생깁니다.
5. 연필, 사각형, 원형, 모두 되돌리기, 되돌리기, 연필 색깔, 연필 두께 순으로 기능이 정렬되어 있습니다. 필요한 기능을 선택하셔서 슬라이드를 꾸미시면 됩니다.
